Lab3 – Task Manager App

This Android application allows users to create, save, and manage tasks with deadlines and notes.

Features

Add tasks with title, deadline, and optional notes

Save tasks locally using SharedPreferences or SQLite

View saved tasks in a list

Tap a task to view its details

Long-press a task to delete it

Simple, clean UI inspired by wellness app designs

Technologies Used

Java

Android SDK

RecyclerView

SharedPreferences & SQLite
