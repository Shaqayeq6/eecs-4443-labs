## Description
A Flutter app that displays a scrollable list of items using `ListView.builder`.  
Selecting an item navigates to a detail screen showing full information.

## Features
- Dynamic list with 15 hardcoded items  
- Custom list item UI  
- Navigation to detail screen  
- Responsive layout (portrait & landscape)
